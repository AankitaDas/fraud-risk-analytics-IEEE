## 📌 Business Problem
Financial institutions face significant losses due to fraudulent transactions. 
This project builds an end-to-end fraud detection and risk scoring platform using real-world IEEE-CIS transaction data.

---

## 🛠 Tech Stack
- PostgreSQL (Data Warehousing & Feature Engineering)
- Python (Pandas, Scikit-Learn, ML Modeling)
- Power BI (Executive Risk Dashboard)
- SQL (CTEs, Window Functions, Aggregations)

---

## 📊 Dataset
- IEEE-CIS Fraud Detection Dataset (Kaggle)
- 590K+ transaction records
- Transaction & Identity data merged for enriched modeling

---

## ⚙️ Project Architecture
1. Raw CSV ingestion into PostgreSQL staging tables
2. Data cleaning & feature engineering using SQL
3. ML model development for fraud probability prediction
4. Risk scoring framework (Low / Medium / High Risk)
5. Executive dashboard built in Power BI

---

## 🤖 Machine Learning
- Supervised classification model
- Feature selection & preprocessing
- Fraud probability scoring per transaction
- Risk categorization logic implemented

---

## 📈 Key Insights
- Identified high-risk device & payment combinations
- Built fraud probability scoring system
- Designed decision-based action framework (Approve / OTP / Block)
- Created KPI dashboard for fraud monitoring
